What is happening with the "for each item" loop?

The script counts how many times "i" appears in the word. The for loop iterates through the word "Mississippi", which has been split into a list by letter. The answer is 4.
The for loop iterates through the items of the list created by split (Mississippi) by (letter). With each iteration of the loop, the variable item becomes the next letter of the list.
