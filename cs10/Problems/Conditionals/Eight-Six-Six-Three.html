"If" statements execute code in them if their given condition is true. "If else" statements do the same thing, except they also have code that is executed only if their given condition is false.

Take a look at the following bunch of "If else" statements. Fill in the conditions appropriately to reach each of the "say" commands!

How can you reach:

  1. Apple

    1: True

  2. Orchard

    1: False, 2: True, 3: True

  3. Banana

    1: False, 2: True, 3: False

  4. Cat

    1: False, 2: False, 4: True

  5. Dance

    1: False, 2: False, 4: False
