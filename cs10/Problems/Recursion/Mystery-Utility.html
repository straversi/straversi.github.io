Mystery Utility

Another case to solve: pictured below is a list utility. It appears to be recursive. input is a list.

  1. Identify the base case. What is happening?
  2. Identify the recursive case. What is going on?
  3. Finally, what is the function doing?

1. The base case is: If the length of the list is 0, return 0.
2. The recursive case is the "else" case. It returns the result of adding 1 to the result of this function with all items but the first of the list.
3. All together, this function returns the length of the input list!
