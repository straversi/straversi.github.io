Mystery Utility

ANOTHER case to solve: pictured below is a list utility. It appears to be recursive. It's input is a list.

  1. Identify the base case. What is happening?
  2. Identify the recursive case. What is going on?
  3. Finally, what is the function doing?
  1. The base case is: If the length of the list is 1, return its first item.
  2. The recursive case is the "else" case. It returns the result of adding the first item of the list to the result of this function with all items but the first of the list.
  3. All together, this function returns the sum of a list's items! Remind you of a particular Higher Order Function? This same result could be achieved by combine with (_+_) item of (list).
