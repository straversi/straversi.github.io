Projects and Tidbits

I like to make fun things with code, and sometimes I put them online.

  • Typer.js - Create a typing effect completely configurable in HTML.
  • Crossword Builder - Build, play, and share 5x5 crosswords.
  • 3D Oscilloscope - Like the 2D one, but with an extra dimension.

Smaller things

  • Cube that Div - Some CSS that makes a <div> look like a cube.
  • Net js - A little javascript visual toy with nodes that are repelled by your mouse.
  • Music Visualization - Exploration of the javascript music analyzer API.
  • Photo Site - A clean design for a photo site that uses easy-gallery.
  • Easy Gallery - Converts an imgur album into HTML with thumbnails.
  • Pixel Transfer - Try to reconstruct a target image using the pixels from a source image.
  • Newton's Cradle - A delightful loading animation.
  • Web Terminal - A fake terminal for a website.
  • 2048 - A homework project for CS10 students at UC Berkeley.

iOS stuff that's not online

  • Monkey Memory - A game in which you quickly try to memorize the positions of 9 numbers.
  • Typerwriter - Make a typerwriter noise every time you press a key.
  • Stereoscope - Take your own stereoscopic photos.
